# 👑 관용어 대결왕 - 실시간 웹 기반 퀴즈 게임 (Firebase 서버리스)

초등학교 6학년 학생들이 관용어의 뜻과 쓰임을 게임 형태로 재미있게 연습할 수 있는 **실시간 팀 대결형 웹 퀴즈 게임**입니다. 

기존 백엔드 서버 대신 구글의 **Firebase Realtime Database**를 활용한 **완전 서버리스(Serverless)** 아키텍처로 구현되어, 복잡한 서버 호스팅 없이 100% 무료로 손쉽게 배포하고 영구적으로 운영할 수 있습니다.

---

## 1. 주요 기능 설명 (Key Features)

1. **Firebase 실시간 동기화 (Serverless Realtime)**
   - 교사의 화면 제어(게임 시작, 다음 문제 이동, 정답 및 해설 공개, 최종 결과 집계)에 맞춰 모든 참여 학생들의 화면이 실시간으로 자동 전환됩니다.
   - 학생들이 정답을 제출하면 교사 대시보드 화면에 실시간 제출 인원수(`제출자: N / 전체: M`)가 표시됩니다.

2. **자동 팀 배정 시스템**
   - 참여 학생들은 로그인이나 회원가입 없이 닉네임과 참여 코드만으로 간단히 입장합니다.
   - 입장 시 각 팀의 인원수를 실시간 트랜잭션으로 연산하여 균등하게 팀을 배정합니다. (청룡팀, 백호팀, 주작팀, 현무팀)

3. **고도화된 점수 및 보너스 시스템**
   - **난이도 기본 점수**: 쉬움(100점), 보통(150점), 어려움(200점)
   - **스피드 보너스**: 10초 이내 정답 제출 시 +30점, 20초 이내 정답 제출 시 +10점 추가 부여
   - **연속 정답 보너스**: 3문제 연속으로 정답을 맞힌 경우 스트릭 보너스 +50점 추가 부여
   - **오답**: 0점 처리 및 스트릭 횟수 0으로 초기화

4. **학습 효과 극대화 및 오답 피드백**
   - 교사가 정답을 공개하면, 학생 화면에 정오답 여부와 더불어 관용어의 **의미 설명(뜻)** 및 **문맥 예문**이 즉시 노출됩니다.
   - 게임이 끝난 후 제공되는 최종 결과 화면에서는 **우승팀 시상**, **개인별 순위**, **정답률 60% 미만의 많이 틀린 관용어 분석**, **오늘 공부한 관용어 사전식 핵심 정리**가 제공됩니다.

5. **자동 문제 시딩 (Auto Seeding)**
   - 데이터베이스가 비어있는 최초 구동 시, 초등학교 6학년 필수 관용어 6문제를 데이터베이스에 자동으로 삽입하여 즉각적인 게임 진행이 가능합니다.

## 2. 로컬 실행 방법 (How to Run Locally)

로컬 환경에 **Node.js(v18 이상)** 및 **npm**이 설치되어 있는지 확인 후 아래 명령어를 터미널에서 차례로 실행합니다.

### 1단계: 패키지 설치
프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 필요한 의존성 라이브러리를 설치합니다.
```bash
npm install
```

### 2단계: 로컬 개발 서버 실행
Vite 로컬 웹서버를 실행합니다.
```bash
npm run dev
```
- 실행 후 터미널에 나타나는 주소(기본값: `http://localhost:5173`)로 접속합니다.

## 3. 웹사이트 배포 방법 (How to Deploy)

이 프로젝트는 서버리스 구조이므로 **Vercel** 또는 **Firebase Hosting**을 통해 단 몇 분 만에 무료로 배포할 수 있습니다.

### 방법 A: Vercel을 이용한 초간단 배포 (추천)
1. [Vercel 공식 사이트](https://vercel.com/)에 회원가입 후 로그인합니다. (GitHub 계정 연동 추천)
2. **[Add New] ➔ [Project]**를 클릭합니다.
3. 내 깃허브 저장소(Repository) 중 `idiom_battle_king`을 가져옵니다(Import).
4. 빌드 설정은 기본값으로 둔 채로 **[Deploy]** 버튼만 누르면 배포가 시작됩니다.
5. 약 1분 후 발급되는 웹사이트 주소(예: `https://idiom-quiz-king.vercel.app`)를 교사와 학생 컴퓨터에서 열어 사용합니다.

### 방법 B: Firebase Hosting을 이용한 배포
1. 터미널을 열고 Firebase CLI 도구를 설치합니다.
   ```bash
   npm install -g firebase-tools
   ```
2. Firebase 계정에 로그인합니다.
   ```bash
   firebase login
   ```
3. 프로젝트를 초기화합니다.
   ```bash
   firebase init hosting
   ```
   - *실행 시 아래와 같이 답변을 선택합니다.*
     - **Project Setup**: `Use an existing project` 선택 후 본인의 프로젝트(`idiom-quiz-king` 등) 선택
     - **Public Directory**: `dist` 입력 (Vite 빌드 폴더)
     - **Single-page app**: `Yes` 선택
     - **Automatic builds with GitHub**: `No` 선택 (필요 시 Yes)
4. React 프로젝트를 빌드한 후 배포 명령어를 내립니다.
   ```bash
   npm run build
   firebase deploy --only hosting
   ```
5. 화면에 표시되는 `Hosting URL`(예: `https://idiom-quiz-king.web.app`) 주소로 접속해 실행합니다.
