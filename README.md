# 👑 관용어 대결왕 - 실시간 웹 기반 퀴즈 게임

초등학교 6학년 학생들이 관용어의 뜻과 쓰임을 게임 형태로 재미있게 연습할 수 있는 **실시간 팀 대결형 웹 퀴즈 게임**입니다. 교사가 방을 만들고 진행을 이끌며, 학생들은 Chromebook이나 태블릿 기기를 이용해 로그인 없이 참여 코드와 닉네임만으로 바로 입장하여 참여합니다.

---

## 1. 폴더 구조 (Folder Structure)

```text
idiom_battle_king/
├── package.json               # 프로젝트 전체 관리 및 동시 실행 설정
├── database.sqlite            # SQLite3 데이터베이스 파일 (자동 생성됨)
├── server/                    # Express + Socket.io 백엔드 서버
│   ├── package.json
│   ├── server.js              # 백엔드 엔트리포인트 및 웹소켓 연결 처리
│   ├── config/
│   │   └── db.js              # SQLite DB 연결 및 기본 데이터 시딩
│   ├── controllers/
│   │   ├── gameController.js  # 방 관리, 입장, 채점, 스트릭, 최종 결과 계산 등
│   │   └── questionController.js # 문제 CRUD 관리
│   └── routes/
│       ├── gameRoutes.js
│       └── questionRoutes.js
└── client/                    # Vite + React 프론트엔드 클라이언트
    ├── package.json
    ├── vite.config.js
    ├── index.html             # 메인 HTML 템플릿
    └── src/
        ├── main.jsx
        ├── App.jsx            # 전체 페이지 라우팅 및 웹소켓 상위 제어
        ├── index.css          # 퀴즈쇼 테마 CSS 디자인 시스템
        ├── api/               # API 통신 함수군 (gameApi, questionApi, scoreApi)
        ├── hooks/             # 커스텀 훅 (useTimer, useGameRoom, useRealtimeScore)
        ├── components/        # 공통 UI 컴포넌트 (Header, QuizCard, ScoreBoard 등)
        └── pages/             # 주요 화면 페이지 (Lobby, Quiz, Results 등)
```

## 2. 주요 기능 설명 (Main Features)

1. **실시간 웹소켓 동기화 (Socket.io)**
   - 교사의 대시보드 조작(게임 시작, 다음 문제 이동, 정답 및 해설 공개, 최종 결과 집계)에 맞춰 모든 참여 학생의 단말 화면이 실시간으로 자동 연동됩니다.
   - 학생들이 정답을 제출하면 교사 화면에 실시간 제출 인원수(`제출자: N / 전체: M`)가 표시됩니다.

2. **자동 팀 배정 시스템**
   - 참여 학생들은 별도의 가입 없이 닉네임과 코드만으로 입장합니다.
   - 입장 시 각 팀의 학생 인원수를 자동으로 계산하여 균등하게 팀을 자동 배정합니다. (청룡팀, 백호팀, 주작팀, 현무팀)

3. **고도화된 점수 및 보너스 시스템**
   - **난이도 기본 점수**: 쉬움(100점), 보통(150점), 어려움(200점)
   - **스피드 보너스**: 10초 이내 정답 제출 시 +30점, 20초 이내 정답 제출 시 +10점 추가 부여
   - **연속 정답 보너스**: 3문제 연속으로 정답을 맞힌 경우 스트릭 보너스 +50점 추가 부여
   - **오답**: 0점 처리 및 스트릭 횟수 0으로 초기화

4. **학습 효과 극대화 및 오답 피드백**
   - 교사가 정답을 공개하면, 학생 화면에 정오답 여부와 더불어 관용어의 **의미 설명(뜻)** 및 **문맥 예문**이 즉시 노출됩니다.
   - 게임이 끝난 후 제공되는 최종 결과 화면에서는 **우승팀 시상**, **개인별 순위**, **정답률 60% 미만의 많이 틀린 관용어 분석**, **오늘 공부한 관용어 사전식 핵심 정리**가 일목요연하게 정리되어 복습 학습을 지원합니다.

5. **문제 은행 관리 기능 (CRUD)**
   - 교사는 대시보드 혹은 별도 메뉴를 통해 관용어 퀴즈 문항을 새로 등록하고, 수정하거나 삭제하여 수업의 난이도와 범위를 손쉽게 조절할 수 있습니다.

## 3. 실행 방법 (How to Run)

로컬 환경에 **Node.js(v18 이상)** 및 **npm**이 설치되어 있는지 확인 후 아래 단계를 따라 프로젝트를 시작하세요.

### 1단계: 의존성 패키지 설치
프로젝트 루트 디렉토리(`idiom_battle_king`)에서 아래 명령어를 실행하여 루트 및 하위(서버, 클라이언트)의 패키지를 설치합니다.
```bash
# 1. 루트 및 동시 실행을 위한 concurrently 설치
npm install

# 2. 서버 및 클라이언트 패키지 일괄 설치
npm run install-all
```

### 2단계: 개발 서버 통합 실행
아래 단계를 실행하면 백엔드 서버(포트 `5000`)와 프론트엔드 개발 서버(포트 `5173`)가 하나의 터미널에서 동시에 실행됩니다.
```bash
npm run dev
```

### 3단계: 웹 브라우저 접속 및 테스트
1. **교사용 (게임 개설)**:
   - 웹 브라우저로 `http://localhost:5173`에 접속합니다.
   - **선생님용 대시보드**를 클릭하고 게임방 제목(예: `6학년 3반 관용어 대결왕`)과 대결할 팀 수(2~4개)를 입력하여 방을 개설합니다.
   - 화면에 표시되는 **6자리 참여 코드**를 확인합니다.

2. **학생용 (게임 참여)**:
   - 별도의 브라우저 창이나 태블릿, 스마트폰 등에서 `http://localhost:5173`에 접속합니다.
   - **학생으로 입장하기**를 클릭하고 교사가 발급한 6자리 코드와 닉네임을 입력하여 입장합니다.
   - 입장 후 대기방에서 팀 배정을 확인하고, 교사가 게임을 시작하기를 기다립니다.

3. **게임 시작 및 퀴즈 진행**:
   - 교사 화면에서 **게임 시작하기**를 클릭하면 모든 학생 화면이 1번 문제로 실시간 전환됩니다.
   - 학생들은 30초 내에 정답을 고르고 **제출**을 누릅니다.
   - 교사가 **정답 공개하기**를 누르면 해설이 실시간으로 노출되며 점수가 집계됩니다.
   - 마지막 문제까지 진행한 뒤 **최종 결과 확인**을 누르면 시상식 및 정오답 분석 결과가 화면에 나타납니다.
